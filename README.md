# MCP Server Manager for VS Code

Browse, install and manage MCP servers for AI agents — directly in VS Code.

## Features

- **📋 Server Catalog** — Browse 30+ MCP servers organized by category
- **⚡ One-Click Install** — Add servers to your `.mcp.json` with one click
- **🔍 Search** — Find the right server for any task
- **📊 Status Bar** — See how many servers you have installed
- **🗑️ Easy Removal** — Remove servers you no longer need

## Usage

1. Open the **MCP Servers** panel in the sidebar
2. Browse available servers by category
3. Click on a server to see details
4. Use `Ctrl+Shift+P` → `MCP: Browse Servers` for quick search

## Commands

| Command | Description |
|---------|-------------|
| `MCP: Browse Servers` | Search and install servers |
| `MCP: Install Server` | Add a new MCP server |
| `MCP: Remove Server` | Remove an installed server |
| `MCP: Show Installed` | List all installed servers |
| `MCP: Refresh Catalog` | Refresh the server catalog |

## Available Server Categories

- **Crypto & DeFi** — Solana blockchain data
- **Weather & Environment** — Global weather, air quality
- **Science & Research** — NASA, agriculture, academic papers
- **Business & Finance** — EU company data, financial filings
- **Security** — CVE vulnerabilities, threat intel
- **Health & Medical** — WHO data, drug databases
- **Legal** — Court decisions, case law
- **AI & Development** — LLM benchmarks, comparisons
- **Agent Infrastructure** — Memory, payments, optimization
